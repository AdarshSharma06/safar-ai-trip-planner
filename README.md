# Safar AI Trip Planner

A modular backend system for AI-powered trip planning.

## Current Features
- JWT Authentication
- Role-based authorization
- Trip creation & ownership enforcement
- Itinerary modeling
- Secure REST APIs
- Global exception handling

## Tech Stack
- Spring Boot
- Spring Security (JWT)
- PostgreSQL
- JPA / Hibernate
